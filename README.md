# Blood Donation Management System

## Project Description
The Blood Donation Management System is a web application built using the MERN stack. It helps manage blood donor information efficiently. 
Users can add, update, delete, and view donor details in an organized way.

## Technologies Used
- React.js for frontend user interface
- Tailwind CSS and DaisyUI for styling
- Node.js for backend runtime
- Express.js for server and API development
- MongoDB for database management
- Axios for API requests

## Features
- Add new blood donor information
- View the list of all donors
- Update donor details
- Delete donor records
- Manage donation history
- Responsive and user-friendly interface

## Project Structure
- Frontend: React application for user interface  
- Backend: Node.js and Express.js API  
- Database: MongoDB for storing donor data
